# Jeremy Hart Portfolio Template

A clean static professional portfolio website designed for GitHub Pages, Netlify, or Vercel.

## How to use

1. Open `index.html` in your browser to preview it.
2. Edit the text directly in `index.html`.
3. Edit colors and layout in `styles.css`.
4. Add your resume as:
   `assets/Jeremy-Hart-Resume.pdf`
5. Replace the headshot placeholder later with an image.

## Free hosting options

### GitHub Pages
Upload these files to a GitHub repository and enable Pages in the repository settings.

### Netlify
Drag and drop the full folder into Netlify.

### Vercel
Import the GitHub repository into Vercel.
